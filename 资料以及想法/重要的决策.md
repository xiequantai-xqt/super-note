# 分散性目标VS月度规划

|              | 分散性目标（零号规划分化）           | 周计划（年度规划分化出去）                                   |
| ------------ | ------------------------------------ | ------------------------------------------------------------ |
| 灵活性       | 每一个目标的周期都是固定的，灵活性差 | 灵活性更强                                                   |
| 时间成本     | 每天需要打卡的目标比较多，时间成本大 | 只需要管理一个目标，时间成本可控                             |
| 计划的完整性 | 能保持目标的绝对完整性               | 不能保证目标的完整性，不过，可以通过设置一个“目标总览”解决目标不完整的问题 |

我的选择：周计划（年度规划分化出去），个别的计划可以设置独立目标。
