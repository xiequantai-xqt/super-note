# 使用cmd

```sh
# 进入D盘
d:

# 查看D盘下面所有的文件夹
dir

# 创建文件夹cmd1
md cmd1

# 删除文件夹team1（空文件情况）
rd team1
# 如果team1文件夹里面有文件
rd /s /q "team1"

```

- `/s`：递归删除文件夹及其所有子内容（包括文件）。
- `/q`：静默模式（无需确认）。
