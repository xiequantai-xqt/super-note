# 使用cmd

```sh
# 进入D盘
d:

# 查看D盘下面所有的文件夹
dir

# 创建文件夹cmd1
md cmd1

# 删除文件夹team1（空文件情况）
rd team1
# 如果team1文件夹里面有文件
rd /s /q "team1"

```

- `/s`：递归删除文件夹及其所有子内容（包括文件）。
- `/q`：静默模式（无需确认）。

## 开发步骤

Java程序开发三步骤：编写、编译、运行。

- 将Java代码编写到扩展名为.java的源文件中
- 通过 javac.exe 命令对该 Java 文件进行编译，生成一个或者多个字节码文件
- 通过 java.exe 命令对生成的class文件进行运行

![](https://educt-files.oss-cn-shenzhen.aliyuncs.com/3f89db62-e8a5-4f97-b124-0333e46f46a9.png)

源代码：

![](https://educt-files.oss-cn-shenzhen.aliyuncs.com/378d5418-17f5-4a6e-b9a0-7dd2003194ab.png)

> 1. class： 关键字，表示“类”，后面跟着类名。
> 2. main()方法的格式是固定的，也就是 `public static void main(String[] args)` ，表示程序的入口。
> 3. 如果main方法非要有一些变化，只能变化String[] args结构。可以写成：方式一，String args[]。 方式二：String[] a。
> 4. 从控制台输出数据的操作
>    1. System.out.println()：输出数据之后，会在控制台换行
>    2. System.out.print()：输出数据后，不换行
> 5. 一个源文件中可以声明多个类，但是最多只能有一个类使用public进行声明，且要求声明为public的类的类名与源文件名相同。

# 注释

Java的注释和JavaScript一样：

- 单行注释：`//`
- 多行注释：`/**/` ，注意，多行注释不能嵌套使用
- 文档注释：`/** 这是文档注释 */`
